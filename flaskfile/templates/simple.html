Tkinter episode 1: basic hello world program

First You must import tkinter with this statement.

from tkinter import *

Then you must create the window by refering to this Attribute of tkinter(Tk).

from tkinter import *

window = Tk()

Then you can try yourself, and then you can see that the window does'nt appear.

Thats likely because, You have created the window but did not run it.

So, For running it, You can add this statement with the attribute at the last(mainloop).

from tkinter import *

window = Tk()

window.mainloop()

Now you can see that your window will appear.

Now we will add some contents/widgets in the MainWindow.

our first widget will be a label(text).

for That, you can first create the label

if you try to remember, that in the first we created the window and made it run, as the same here, we need it to be visible.

Displaying text is the main function of it, so I passed in text in it(Label).

So, after creating that Label, We can add this attribute(pack) to it.

from tkinter import *

window = Tk()

window.mainloop()

# widgets go here....

# remember that for craeting the Label, first parameter it takes is, where to create, i.e. the Window you created.

# other all parameters are optional.

label = Label(window, text='This is a Label')

# We will now pack it.

label.pack()
